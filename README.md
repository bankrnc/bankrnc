## Projects

**StridePilot** — AI-Powered Running Training Platform  
[stride-pilot.vercel.app](https://stride-pilot.vercel.app) · React · TypeScript · Node.js · Express · PostgreSQL · Claude API

A fullstack web application that generates personalized 4-week running programs using AI,
based on user goals, fitness level, target pace, and heart rate zones.

Key implementation details:
- JWT authentication with httpOnly cookies, session management, and route protection
- Performance scoring engine comparing planned vs. actual training data
- Analytics dashboard covering HR zone distribution, session types, and weekly volume
- Schema validation with Zod; MVC architecture with centralized error handling
- Deployed via Vercel (frontend), Railway (backend), and Neon (PostgreSQL)
